# Aplikasi Android 2 Activity
Aplikasi ini terdiri dari dua halaman, di halaman pertama terdapat penjelasan singkat sejarah android dan di halaman kedua berisi tentang arsitektur android dan informasi device.

## Sejarah Android
Didirikan tahun 2003 oleh empat orang bernama Andy Rubin, Rich Miner, Nick Sears, dan Chris White.
Pada tahun 2005, android diakuisisi oleh Google dan sejak tahun 2008 android didistribusikan melalui Android Market dengan versi terbaru hingga saat ini adalah Android 15.

## Arsitektur Android
Android memiliki 6 lapisan utama:
1. Linux Kernel
2. Hardware Abstraction Layer (HAL)
3. Native Libraries
4. Android Runtime (ART)
5. Application Framework
6. Applications

## Informasi Device
Dalam aplikasi ini, informasi device yang ditampilkan meliputi model, manufactur, brand, baterai, user, type, dan host
