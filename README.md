# Aplikasi Android Sederhana 2 Activity
Aplikasi ini terdiri dari dua halaman (activity), di halaman pertama terdapat penjelasan singkat sejarah android dan di halaman kedua berisi tentang arsitektur android dan informasi device.

## Sejarah Android
Didirikan tahun 2003 oleh empat orang bernama Andy Rubin, Rich Miner, Nick Sears, dan Chris White.
Pada tahun 2005, android diakuisisi oleh Google dan sejak tahun 2008 android didistribusikan melalui Android Market dengan versi terbaru hingga saat ini adalah Android 15.

## Arsitektur Android
Berdasarkan arsitekturnya, android memiliki 6 lapisan, yaitu:
1. Linux Kernel
2. Hardware Abstraction Layer (HAL)
3. Native Libraries & Android Runtime (ART)
4. Java API Framework
5. Applications

## Informasi Device
Dalam aplikasi ini, informasi device yang ditampilkan meliputi versi android, model, manufaktur, brand, baterai, kernel, network status, user, type, dan host.
